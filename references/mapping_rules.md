# 映射规则 v1.0

> 心智模型特征 → 数据分析方法的映射表

## 基础映射

| 思维特征关键词 | 分析方法 | 说明 |
|--------------|---------|------|
| 归因、贡献、谁的责任、拖累 | delta_attribution | 找出变化的主要贡献者 |
| 反转、反过来想、避免 | 归因 + 末位排名 | 先找"谁在拖后腿" |
| 系统、脆弱性、崩溃点 | sensitivity_analysis + anomaly_detection | 找系统薄弱环节 |
| 长期、趋势、周期 | stl_decomposition | 看长期趋势而非短期波动 |
| 概率、赌注、风险 | ab_test + power_analysis | 用实验验证而非猜测 |
| 分层、不同类型、差异化 | kmeans_clustering + stratified_comparison | 先分群再对比 |
| 杠杆、20/80、关键少数 | pareto_analysis | 找出最重要的20% |
| 因果、真实效果、净影响 | did + psm_matching | 控制混淆变量看真实因果 |
| 漏斗、转化、流失 | funnel_conversion | 找流失瓶颈 |
| 对比、谁更好、差异 | ttest / anova | 统计显著性对比 |

## 人物常见视角映射

### 芒格视角
- **反转思维** → 先做归因找负面贡献者，再末位排名亮灯
- **多元思维模型** → 一个数据集跑多种分析方法（归因+时序+聚类）
- **能力圈** → 标注分析中"超出数据范围的推断"

### 塔勒布视角
- **反脆弱** → 做敏感性分析，找哪些因素让系统更脆弱/更强壮
- **尾部风险** → 异常检测 + 极端值分析
- **skin in the game** → 检查数据中"有利益相关"和"没有利益相关"的群体差异

### 费曼视角
- **第一性原理** → 把复杂指标拆解到最基本构成（如单位成本 = 人工成本/操作量 + 固定成本分摊/操作量）
- **简化解释** → 每个分析结论用大白话重述一遍
- **怀疑精神** → 对每个结论标注"这个推断有多可靠"

### 彼得·蒂尔视角
- **从0到1** → 找"完全新的做法"vs"只是做得更好"的区分
- **垄断思维** → 帕累托分析，找是否存在"赢家通吃"格局
- **逆向问题** → "什么真理很少人同意你？"→ 找数据中的反共识发现
